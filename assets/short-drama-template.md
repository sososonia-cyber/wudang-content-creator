# 短剧剧本模板（1-3分钟）

## 剧本格式说明

```
【剧集信息】
剧名：
集数：第X集
时长：X分钟
类型：（喜剧/治愈/悬疑/武侠等）

【人物介绍】
角色名：（年龄/身份/性格特点）

【场景列表】
场景X：地点 - 时间

【正文】
场景编号
景别：（远景/全景/中景/近景/特写）
机位：（固定/移动/手持等）
画面描述：
对白/旁白：
音效/音乐：
```

## 1分钟短剧模板

### 模板A：治愈系（都市焦虑）

```
【剧集信息】
剧名：《山中有答案》
集数：第1集
时长：1分钟
类型：治愈系

【人物介绍】
小雨：（25岁，职场白领，焦虑疲惫）
清虚道长：（60岁，武当山道长，智慧平和）

【场景列表】
场景1：写字楼 - 晚上
场景2：武当山道观 - 白天
场景3：金顶 - 日出

【正文】
场景1 写字楼
景别：近景
机位：固定
画面描述：小雨对着电脑，黑眼圈，疲惫地揉着太阳穴。窗外是城市夜景。
对白：
小雨（自言自语）：又要加班到半夜...
音效：键盘敲击声，空调声

（转场：快速剪辑城市焦虑画面，配合压抑音乐）

场景2 武当山道观
景别：全景→中景
机位：移动
画面描述：小雨拖着行李箱，迷茫地站在道观门口。清虚道长正在扫落叶。
对白：
小雨：道长，我很累，不知道该怎么办。
清虚道长（微笑）：来，先喝一杯茶。
音效：风吹树叶声，远处钟声

（转场：柔和音乐起）

场景3 金顶
景别：远景→特写
机位：固定→缓慢推近
画面描述：日出云海，小雨和道长并排站立。小雨深呼吸，露出微笑。
对白：
清虚道长：答案不在外面，在心里。
小雨（释然）：我明白了。
字幕：当你迷茫时，不妨去山里走走

【结尾】
字幕：《山中有答案》每周更新
引导：关注看完整版
```

### 模板B：喜剧（古今反差）

```
【剧集信息】
剧名：《道长下山》
集数：第1集
时长：1分钟
类型：喜剧

【人物介绍】
云鹤道长：（40岁，武当山道长，严肃但反差萌）
小明：（20岁，大学生，热情活泼）

【场景列表】
场景1：武当山 - 白天
场景2：城市奶茶店 - 白天
场景3：网吧 - 晚上

【正文】
场景1 武当山
景别：全景
机位：固定
画面描述：云鹤道长仙风道骨，正在练剑。
对白：
小明（画外音）：道长，我带你去见见世面！

场景2 奶茶店
景别：中景
机位：手持
画面描述：道长看着菜单，一脸懵逼。
对白：
云鹤道长：这...珍珠奶茶是道茶吗？
小明：不是，这是...额...甜的！
云鹤道长（喝一口）：甜！甚好！再来一杯！

场景3 网吧
景别：近景
机位：固定
画面描述：道长戴着耳机，专注打游戏。
对白：
云鹤道长：这游戏，比打坐有意思！
小明（无奈）：道长，您慢点...

（道长五杀，全场欢呼）

【结尾】
云鹤道长（对镜头）：下山真好玩！
字幕：道长下山记，每周更新
```

## 2分钟短剧模板

### 模板A：武侠风（传承）

```
【剧集信息】
剧名：《武当少年》
集数：第1集
时长：2分钟
类型：武侠/励志

【人物介绍】
阿杰：（18岁，城市少年，性格叛逆但善良）
青松道长：（55岁，武当山武术教练，严厉但关怀）
师兄：（25岁，道观学徒，友善）

【场景列表】
场景1：城市街道 - 白天
场景2：武当山练武场 - 白天
场景3：宿舍 - 晚上
场景4：山道 - 清晨

【正文】
场景1 城市街道
景别：中景
机位：跟随
画面描述：阿杰和混混打架，被警察带走。父母失望。
对白：
父亲：你就不能学好点吗！
阿杰（倔强）：我不需要你们管！
旁白：这个叛逆少年，即将开始一段改变人生的旅程。

场景2 武当山练武场
景别：全景
机位：固定
画面描述：阿杰被送到武当山，青松道长正在教其他学员太极。
对白：
青松道长：既然来了，就要守规矩。
阿杰（不屑）：太极？老头乐？
青松道长（平静）：明天开始，站马步。

场景3 宿舍
景别：近景
机位：固定
画面描述：阿杰累瘫在床上，师兄递来毛巾。
对白：
师兄：第一天都这样，坚持住。
阿杰：我为什么要受这个罪...
师兄：你来这里，不就是为了改变吗？
阿杰（沉默）

场景4 山道
景别：远景→特写
机位：移动
画面描述：清晨，阿杰独自在山道上练习。从笨拙到逐渐流畅。
音乐：激昂的武侠音乐
对白：
阿杰（内心独白）：原来，真正的力量不是打败别人，而是控制自己。

（阿杰完成一套动作，道长远远看着，点头微笑）

【结尾】
字幕：真正的强大，从控制自己开始
下集预告：阿杰的第一场比赛
```

### 模板B：爱情（相遇）

```
【剧集信息】
剧名：《云上有你》
集数：第1集
时长：2分钟
类型：爱情/治愈

【人物介绍】
林夕：（24岁，摄影师，文艺气质）
墨言：（26岁，武当山道士，清朗温和）

【场景列表】
场景1：武当山金顶 - 日出
场景2：山道 - 白天
场景3：道观茶室 - 傍晚

【正文】
场景1 金顶日出
景别：远景
机位：固定
画面描述：壮丽的日出云海。林夕在拍照，墨言在晨练太极。
对白：
林夕（惊叹）：太美了！
（相机快门声）
墨言（被声音吸引，转头）

（两人目光相遇，有些尴尬）
林夕：对不起，打扰你了。
墨言（微笑）：无妨。你拍得很好。

场景2 山道
景别：中景
机位：跟随
画面描述：林夕下山时崴脚，墨言恰好路过。
对白：
林夕：啊！
墨言（快步上前）：小心！
（扶住林夕）
墨言：扭到了？我懂一点推拿。
林夕：那...麻烦你了。

（墨言帮林夕处理脚踝，动作轻柔）
对白：
墨言：你是来旅游的？
林夕：嗯，来采风。你呢？
墨言：我住在这里。
林夕（惊讶）：住在山上？
墨言：是啊，清净。

场景3 道观茶室
景别：近景
机位：固定
画面描述：傍晚，墨言邀请林夕喝茶。夕阳透过窗户。
对白：
墨言：这是我们自己采的茶。
林夕（品尝）：很清香。
墨言：你为什么要拍武当山？
林夕：因为...我觉得这里有故事。
墨言（微笑）：每个人都有故事。

（两人相视一笑，音乐起）

【结尾】
字幕：有些相遇，是命中注定
下集预告：林夕要走了，墨言会挽留吗？
```

## 3分钟短剧模板

### 模板A：悬疑（寻找）

```
【剧集信息】
剧名：《武当秘录》
集数：第1集
时长：3分钟
类型：悬疑/武侠

【人物介绍】
沈墨：（30岁，文物调查员，冷静理性）
慧清道长：（70岁，武当山藏书阁守阁人，神秘）
神秘人：（身份不明，只露背影）

【场景列表】
场景1：武当山藏书阁 - 白天
场景2：山路 - 黄昏
场景3：密室 - 晚上
场景4：金顶 - 深夜

【正文】
场景1 藏书阁
景别：中景
机位：手持（营造紧张感）
画面描述：沈墨在古籍中寻找线索，慧清道长远远观察。
对白：
沈墨（自言自语）：这本《武当秘录》记载了什么...
慧清道长：年轻人，有些东西，不该被找到。
沈墨：道长，我只是在调查文物失窃案。
慧清道长：失窃？还是...被取走？

场景2 山路
景别：远景→中景
机位：跟随
画面描述：沈墨独自走在山路上，发现有人跟踪。加快脚步。
音效：脚步声，风声

（突然，一个黑影闪过）

场景3 密室
景别：近景
机位：固定
画面描述：沈墨发现一处隐藏密室，墙上是古老的壁画。
对白：
沈墨（惊讶）：这是...明代的...
（手机拍摄）

（突然灯光熄灭，手机被抢走）
神秘人（画外音，变声）：你不该来这里。

场景4 金顶
景别：全景
机位：缓慢环绕
画面描述：深夜，沈墨站在金顶，俯瞰山下灯火。
对白：
沈墨（内心独白）：武当山，到底藏着什么秘密？

（手机响起，一条匿名短信："想知道真相，明晚子时，紫霄宫见。"）

【结尾】
字幕：真相，即将浮出水面
下集预告：紫霄宫之约，是福是祸？
```

## 剧本写作技巧

### 结构设计

**1分钟短剧**：
- 起（15秒）：快速建立人物和场景
- 承（30秒）：展开情节或冲突
- 转（10秒）：转折或高潮
- 合（5秒）：结尾或悬念

**2-3分钟短剧**：
- 第一幕（30秒）：建立世界观和人物
- 第二幕（60-90秒）：发展冲突
- 第三幕（30秒）：高潮和解决

### 对白写作

- 简洁有力，避免废话
- 符合人物性格
- 每句话都有目的（推动情节/揭示性格/制造冲突）

### 视觉呈现

- 多运用画面讲故事，减少对白
- 利用武当山景观营造氛围
- 服装道具要符合人物身份

### 节奏把控

- 关键情节放慢，给观众反应时间
- 转场要流畅
- 结尾要有记忆点

### 成本考虑

- 控制场景数量（建议3-5个）
- 演员数量不宜过多（2-4人）
- 充分利用武当山免费场景
