# 短视频脚本模板（15秒/30秒/60秒）

## 脚本格式说明

```
【时长】总时长
【BGM】背景音乐建议
【镜头】分镜描述
【时长】该镜头时长
【画面】画面内容
【旁白/对白】台词
【字幕】屏幕文字
【音效】特效音
```

## 30秒脚本模板（三幕式结构 - 推荐）

### 模板A：三幕式故事（三条10秒AI视频拼接）

**创作思路**：使用AI生成三条10秒视频，分别对应三幕结构，剪辑拼接成30秒完整视频。

**第一幕（10秒）：场景引入**
```
【时长】10秒
【AI提示词】武当山清晨，薄雾缭绕，金顶在云雾中若隐若现，远山层叠，宁静祥和的仙境氛围，竖屏视频
【BGM】轻柔的古筝音乐渐入

【镜头1】
【时长】10秒
【画面】航拍视角，从云雾中缓缓推进，露出金顶一角
【字幕】清晨5点，武当山
【旁白】有一种宁静，藏在云深处
【音效】风声，远处钟声
```

**第二幕（10秒）：人物/动作发展**
```
【时长】10秒
【AI提示词】武当山道长身穿白色道袍，在金顶云海前练习太极拳，动作行云流水，衣袂飘飘，竖屏视频
【BGM】音乐节奏加强，加入鼓点

【镜头1】
【时长】10秒
【画面】道长太极动作特写，背景是翻腾的云海
【字幕】跟随自然，顺势而为
【旁白】在这里，时间慢了下来
【音效】衣袂飘动声，呼吸声
```

**第三幕（10秒）：高潮升华**
```
【时长】10秒
【AI提示词】武当山日出，金色阳光穿透云海，普照金顶，道长收势静立，背影剪影，壮丽的仙境画面，竖屏视频
【BGM】音乐达到高潮，然后渐弱

【镜头1】
【时长】10秒
【画面】日出金光，道长剪影，拉远镜头展现全景
【字幕】道法自然，生生不息
【旁白】武当山，等风也等你
【音效】钟声，风声渐弱
```

**剪辑建议**：
- 三条视频之间使用「叠化」或「闪白」转场
- 添加统一的色调滤镜（暖色调/电影感）
- BGM在转场处做淡入淡出处理
- 字幕风格保持一致

## 脚本写作技巧

### 黄金3秒原则

前3秒必须抓住观众注意力：
- 视觉冲击（美景/反差）
- 悬念提问
- 直接展示痛点

### 节奏控制

- **15秒**：快节奏，每3-5秒一个转场
- **30秒**：中快节奏，有起承转合
- **60秒**：可以讲故事，但避免拖沓

### 字幕设计

- 字体醒目，位置居中或下方
- 关键信息加大或变色
- 配合画面节奏出现

### BGM选择

- 国潮内容：国风电子音乐
- 治愈内容：轻音乐、钢琴曲
- 震撼内容：史诗级配乐
- 日常内容：轻快vlog音乐

### 互动设计

在结尾处设计互动：
- 提问引导评论
- 挑战引导参与
- 福利引导关注
