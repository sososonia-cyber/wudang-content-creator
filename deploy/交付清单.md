## 部署步骤 (后续执行)

```bash
# 1. 登录服务器
ssh root@your-server-ip

# 2. 上传部署包
scp -r wudang-cms-deploy root@your-server-ip:/opt/

# 3. 执行部署
cd /opt/wudang-cms-deploy
./scripts/init.sh

# 4. 验证
./scripts/health-check.sh
```
